Git Command Cheat Sheet

Tu guía rápida de comandos esenciales aprendidos en el bootcamp.

1. Configuración Inicial

Lo primero que haces en una PC nueva. Fundamental para que tus commits tengan tu firma.

# Configura tu identidad global
git config --global user.name "Your Name"
git config --global user.email "you@email.com"
# Ajustes de calidad de vida recomendados
git config --global init.defaultBranch main

2. Arrancando Motores

Cómo empezar un proyecto, ya sea desde cero en tu PC o trayéndolo de la nube.

# Convierte la carpeta actual en un repositorio
git init
# Descarga un proyecto existente
git clone [URL_SSH_REPO]

3. El Ciclo Diario

El flujo sagrado de trabajo: trabajar, preparar y guardar. Repetir 100 veces al día.

# Comprueba qué ha cambiado
git status
# Prepara TODOS los archivos cambiados (Stage)
git add .
# Guarda la versión con un mensaje
git commit -m "action verb describing change"

4. Ramas (Universos Paralelos)

Nunca trabajes directo en `main`. Crea espacios seguros para tus experimentos.

# Crea UNA NUEVA rama y te mueve a ella
git checkout -b new-branch-name
# Regresa a la rama principal
git checkout main

5. Nube y Sincronización

Conectando tu PC local con GitHub. Subir tus cambios y bajar los de tus compañeros.

# Conecta tu repo local con el de GitHub
git remote add origin [URL]
# Sube tu rama actual a GitHub (publicar)
git push -u origin branch-name
# Baja los cambios nuevos de la nube
git pull origin main

6. Información e Historial

Comandos para no perderse y ver qué ha pasado antes.

# Historial de commits resumido
git log --oneline
# Verifica la conexión SSH
ssh -T git@github.com
